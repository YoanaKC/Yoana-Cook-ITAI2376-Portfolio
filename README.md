## Featured Project — GridGuard-AI

An autonomous 6-agent AI system that monitors the Texas (ERCOT) power grid in real time and generates executable dispatch recommendations. Built on a production infrastructure developed independently starting October 2025.

**My individual contributions:**
- Automated ERCOT data collection pipeline (18,742+ records)
- SQLite database with 40-feature engineering engine
- Random Forest predictive model (92.8–95.7% accuracy)
- Flask REST API
- 3 specialist CrewAI agents: Renewable Forecaster, Market Analyst, Compliance Officer (RAG)

**Tech stack:** Python, CrewAI, Groq/Llama 3.3 70B, LSTM, Random Forest, ChromaDB, Flask, SQLite
